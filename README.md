# gif2mp4

Converting from an animated GIF to MP4 should be quick and easy.

## installation

```sh
brew tap christianbundy/tap
brew install gif2mp4
```

## usage

```sh
gif2mp4 input [ output ]
```

## example use

```sh
gif2mp4 cat.gif
```

```
gif2mp4 cat.gif kitty.mp4
```

## uninstallation

```sh
brew uninstall gif2mp4
```
